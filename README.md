# 📘 Web Project: Flask + React + TypeScript

## 📝 프로젝트 개요
- **목표:** 게시판, 로그인, 회원가입 기능을 갖춘 풀스택 웹 애플리케이션 개발
- **백엔드:** Flask (Python)
- **프론트엔드:** React + TypeScript
- **패턴:** MVC 아키텍처 적용
- **개발 및 배포:** 제작부터 배포까지 완성 목표
